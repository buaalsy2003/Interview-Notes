### 算法复杂度
- $O$ 渐进上界；$\Omega$ 渐进下界；$\Theta$ 渐进上下界
- 算法复杂度有平均复杂度和最快复杂度，通常情况下指**最坏复杂度**
- 递归复杂度求解: 主定律和递归树
- 主定律：$T(n) <= aT(n/b) + n^d$
 1. $T(b) = O(n^d \log n)$     if $d = \log _ab$
 2. $T(b) = O(n^d)$            if $d > \log _ab$
 3. $T(b) = O(n^{\log _ab})$   if $d > \log _ab$
